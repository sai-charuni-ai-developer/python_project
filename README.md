# 🧮 LangChain Calculator Agent

A beginner-friendly AI calculator agent built using LangChain in Python.

## ✨ Features
- Add, subtract, multiply, divide
- Percentage calculations
- Square root
- Power / exponents
- Supports natural language like "add 10 and 25" or "15 percent of 200"

## 🛠️ Built With
- Python 3
- LangChain

## ⚙️ Installation
pip3 install langchain numexpr

## ▶️ How to Run
python3 calculator_agent.py

## 💬 Example Questions You Can Ask
- add 10 and 25
- 15 percent of 200
- square root of 144
- 2 to the power of 8
- divide 100 by 4
- multiply 6 times 7
- 10 + 20
